# Planning séances HYROX

Préparation pour HYROX Paris Porte de Versailles, cible actuelle : **samedi 12 décembre 2026**.

Ce document reprend le planning généré par l'application pour les **3 configurations badminton** disponibles.

Les séances force, course/moteur et HYROX gardent le même contenu de progression. Ce qui change selon la configuration, c'est surtout le placement dans la semaine pour respecter le badminton et éviter de surcharger les soirs.

Les semaines allégées sont prévues toutes les 4 semaines. Les semaines de vacances restent à renseigner dans les réglages de l'application ; quand une semaine est marquée en vacances, l'application bascule automatiquement en mode maintenance.

## Séances fixes

### Lundi - Repos complet

- Objectif : récupérer vraiment, absorber la charge et protéger le lundi.
- Durée : 0 min.
- RPE cible : 0.
- Fatigué : repos complet, mobilité 8 min si ça fait du bien.
- Normal : repos complet, marche douce possible.
- En forme : repos quand même. L'énergie sert à mieux encaisser la suite.

### Badminton

Selon la configuration choisie, le badminton tombe mercredi + jeudi, mardi + mercredi + jeudi, ou mercredi + jeudi + vendredi.

- Objectif : travailler les appuis, le cardio intermittent et le plaisir sans saboter la récupération.
- Durée cible : 75 min.
- RPE cible : 6-8.
- Fatigué : échauffement long, jeu technique, peu de matchs à haute intensité. Stop si douleur mollet, tendon ou genou.
- Normal : séance normale. 8 à 10 min d'échauffement : squats, fentes, mollets, pas chassés, accélérations.
- En forme : séance normale à intense, sans dépasser 90 min de haute intensité. Hydratation + protéines après.

### Récupération active

La récupération active est prévue le samedi dans la configuration A. Dans les configurations B et C, le samedi est utilisé pour la course/moteur, donc la récupération doit rester douce et opportuniste : marche, mobilité, sommeil, retour au calme après séance.

- Objectif : faire circuler sans ajouter de fatigue.
- Durée cible : 40 min.
- RPE cible : 3-4.
- Fatigué : 15 à 20 min marche facile + respiration.
- Normal : 30 à 45 min zone 2 très facile : vélo, marche inclinée ou mobilité.
- En forme : 45 min zone 2 facile + mobilité longue, pas d'intensité cachée.

## Versions adaptables

Pour chaque séance non repos, l'application garde trois versions :

- **Fatigué** : version courte, technique, RPE plafonné.
- **Normal** : séance prévue.
- **En forme** : option renforcée, seulement si récupération OK.

Règle pratique :

- Si sommeil mauvais, douleurs tendineuses, fatigue nerveuse ou badminton très intense : utiliser la version fatigué.
- Si tout est normal : garder la version normale.
- Si énergie haute depuis plusieurs jours, pas de douleur et semaine non allégée : version en forme possible.

## Notes vacances

Quand une semaine est déclarée en vacances dans les réglages :

- Badminton devient loisir contrôlé.
- Course devient footing vacances + mobilité.
- Force/HYROX devient circuit hôtel léger.
- Objectif : maintenir le rythme sans créer de dette de fatigue.
